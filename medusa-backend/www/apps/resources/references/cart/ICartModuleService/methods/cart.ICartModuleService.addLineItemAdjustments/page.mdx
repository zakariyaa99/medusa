## addLineItemAdjustments(data): Promise&#60;[LineItemAdjustmentDTO](../../../interfaces/cart.LineItemAdjustmentDTO/page.mdx)[]&#62;

### Example

```ts
const lineItemAdjustments =

      code: "10%OFF",
      amount: 3000,
    },
  ])
