# updateLocales - Translation Module Reference

This documentation provides a reference to the `updateLocales` method. This belongs to the Translation Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateLocales(data, sharedContext?): Promise&#60;[LocaleDTO](../../../interfaces/translation.LocaleDTO/page.mdx)&#62;

This method updates an existing locale. The ID should be included in the data object.

### Example

```ts
const locale = await translationModuleService.updateLocales({
  id: "loc_123",
  name: "English (United States)",
})
```

## updateLocales(dataOrOptions, sharedContext?): Promise&#60;[LocaleDTO](../../../interfaces/translation.LocaleDTO/page.mdx)[]&#62;

This method updates existing locales either by ID or by a selector.

### Example

To update locales by their IDs:

```ts
const locales = await translationModuleService.updateLocales([
  {
    id: "loc_123",
    name: "English (United States)",
  },
 {
    id: "loc_321",
    name: "French (France)",
  },
])
```

To update locales by a selector:

```ts
const locales = await translationModuleService.updateLocales({
  selector: {
    code: "en-US",
  },
  data: {
    name: "English (United States)",
  },
})
```
