# dismissRemoteLinkStep - Helper Steps API Reference

This documentation provides a reference to the `dismissRemoteLinkStep` step. It belongs to the `@medusajs/medusa/core-flows` package.

This step removes links between two records of linked data models.

Learn more in the [Link documentation.](https://docs.medusajs.com/learn/fundamentals/module-links/link#dismiss-link).

## Example

```ts
dismissRemoteLinkStep([{
  [Modules.PRODUCT]: {
    product_id: "prod_123",
  },
  blog: {
    post_id: "post_123",
  },
}])
```
