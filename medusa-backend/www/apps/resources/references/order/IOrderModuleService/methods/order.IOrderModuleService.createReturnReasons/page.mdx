# createReturnReasons - Order Module Reference

This documentation provides a reference to the `createReturnReasons` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## createReturnReasons(returnReasonData, sharedContext?): Promise&#60;[OrderReturnReasonDTO](../../../interfaces/order.OrderReturnReasonDTO/page.mdx)&#62;

This method creates a return reason.

### Example

```ts
const returnReason = await orderModuleService.createReturnReasons({
  label: "Damaged",
  value: "damaged"
})
```
