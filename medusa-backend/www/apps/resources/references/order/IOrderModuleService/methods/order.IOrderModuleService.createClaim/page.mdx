# createClaim - Order Module Reference

This documentation provides a reference to the `createClaim` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method creates a claim.

## Example

```ts
const claim = await orderModuleService.createClaim({
  order_id: "123",
  type: "refund"
})
```
