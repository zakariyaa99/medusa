# setOrderShippingMethodAdjustments - Order Module Reference

This documentation provides a reference to the `setOrderShippingMethodAdjustments` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method sets the shipping method adjustments of an order.

## Example

```ts
const shippingMethodAdjustments = await orderModuleService
  .setOrderShippingMethodAdjustments(
    "123",
    [
      {
        shipping_method_id: "123",
        code: "50OFF",
        amount: 5
      },
      {
        id: "321",
        amount: 5
      }
    ]
  )
```
