# cancel - Order Module Reference

This documentation provides a reference to the `cancel` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## cancel(orderId, sharedContext?): Promise&#60;[OrderDTO](../../../interfaces/order.OrderDTO/page.mdx)[]&#62;

This method sets the status of orders to `canceled`.

### Example

```ts
const orders = await orderModuleService.cancel(["123", "321"])
```
