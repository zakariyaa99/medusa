# updateOrderShippingMethods - Order Module Reference

This documentation provides a reference to the `updateOrderShippingMethods` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateOrderShippingMethods(data, sharedContext?): Promise&#60;[OrderShippingMethodDTO](../../../interfaces/order.OrderShippingMethodDTO/page.mdx)[]&#62;

This method updates existing shipping methods. The shipping method IDs are specified in each shipping method object.

### Example

```ts
const shippingMethods = await orderModuleService.updateOrderShippingMethods([{
  id: "123",
  name: "Express Shipping"
}])
```
