# addOrderAction - Order Module Reference

This documentation provides a reference to the `addOrderAction` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## addOrderAction(data, sharedContext?): Promise&#60;[OrderChangeActionDTO](../../../interfaces/order.OrderChangeActionDTO/page.mdx)&#62;

This method creates and adds an action to an order and its change.

### Example

```ts
const orderChangeAction = await orderModuleService.addOrderAction({
  order_id: "123",
  order_change_id: "321",
  action: "ITEM_ADD"
})
```
