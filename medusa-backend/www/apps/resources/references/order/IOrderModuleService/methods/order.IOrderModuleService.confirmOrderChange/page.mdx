# confirmOrderChange - Order Module Reference

This documentation provides a reference to the `confirmOrderChange` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## confirmOrderChange(orderChangeId, sharedContext?): Promise&#60;[OrderChangeReturn](../../../interfaces/order.OrderChangeReturn/page.mdx)&#62;

This method confirms an order change, applying these changes on the associated order.

### Example

```ts
const {
  items,
  shippingMethods
} =  await orderModuleService.confirmOrderChange("123")
```
