# updateOrderItem - Order Module Reference

This documentation provides a reference to the `updateOrderItem` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateOrderItem(selector, data, sharedContext?): Promise&#60;[OrderItemDTO](../../../interfaces/order.OrderItemDTO/page.mdx)[]&#62;

This method updates existing order items matching the specified filters.

### Example

```ts
const orderItems = await orderModuleService.updateOrderItem({
  id: "123"
}, {
  quantity: 2
})
```
