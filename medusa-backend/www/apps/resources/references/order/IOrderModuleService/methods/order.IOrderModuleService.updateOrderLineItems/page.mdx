# updateOrderLineItems - Order Module Reference

This documentation provides a reference to the `updateOrderLineItems` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateOrderLineItems(data, sharedContext?): Promise&#60;[OrderLineItemDTO](../../../interfaces/order.OrderLineItemDTO/page.mdx)[]&#62;

This method updates existing line items. The line item to update is specified by the `selector` property of the first parameter.

### Example

```typescript
const lineItems = await orderModuleService.updateOrderLineItems([
  {
    selector: {
      id: "123"
    },
    data: {
      quantity: 2
    }
  }
])
```
