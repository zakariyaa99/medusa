# createViewConfigurationWorkflow - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: view_configurations](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `createViewConfigurationWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

:::
