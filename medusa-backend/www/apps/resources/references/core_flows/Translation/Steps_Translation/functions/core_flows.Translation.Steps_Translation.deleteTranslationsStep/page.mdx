# deleteTranslationsStep - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: translation](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `deleteTranslationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step deletes one or more translations.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## Example

```ts
const data = deleteTranslationsStep([
  "trans_123",
  "trans_456",
])
```
