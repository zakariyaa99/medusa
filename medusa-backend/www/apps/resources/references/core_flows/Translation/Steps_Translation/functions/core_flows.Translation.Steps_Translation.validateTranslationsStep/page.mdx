# validateTranslationsStep - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: translation](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `validateTranslationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step validates that the translations are supported by the store.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## Example

```ts
const data = validateTranslationsStep([
  {
    reference_id: "prod_123",
    reference: "product",
    locale: "fr-FR",
    translations: { title: "Produit", description: "Description du produit" }
  }
])
```
