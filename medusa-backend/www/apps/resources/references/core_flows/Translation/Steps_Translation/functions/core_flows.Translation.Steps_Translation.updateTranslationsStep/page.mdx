# updateTranslationsStep - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: translation](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `updateTranslationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates translations matching the specified filters or by ID.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## Example

To update translations by their ID:

```ts
const data = updateTranslationsStep({
  translations: [
    { id: "trans_123", translations: { title: "Nouveau titre" } }
  ]
})
```

To update translations matching filters:

```ts
const data = updateTranslationsStep({
  selector: {
    reference_id: "prod_123",
    locale: "fr-FR"
  },
  update: {
    translations: { title: "Nouveau titre" }
  }
})
```
