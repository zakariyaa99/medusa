# updateTranslationsWorkflow - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: translation](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `updateTranslationsWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow updates translations matching the specified filters or IDs. It's used by other
workflows like the [batchTranslationsWorkflow](../core_flows.Translation.Workflows_Translation.batchTranslationsWorkflow/page.mdx) workflow.

You can use this workflow within your own customizations or custom workflows, allowing you
to update translations in your custom flows.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## Examples

To update translations by their IDs:

<CodeTabs group="workflow-exection">
    <CodeTab label="API Route" value="api-route">
    
```ts title="src/api/workflow/route.ts"
import type {
  MedusaRequest,
  MedusaResponse,
} from "@medusajs/framework/http"
import { updateTranslationsWorkflow } from "@medusajs/medusa/core-flows"

export async function POST(
  req: MedusaRequest,
  res: MedusaResponse
) {
  const { result } = await updateTranslationsWorkflow(req.scope)
    .run({
      input: {
        translations: [
          { id: "trans_123", translations: { title: "Nouveau titre" } }
        ]
      }
    })

  res.send(result)
}
```

To update translations matching filters:

<CodeTabs group="workflow-exection">
    <CodeTab label="API Route" value="api-route">
    
```ts title="src/api/workflow/route.ts"
import type {
  MedusaRequest,
  MedusaResponse,
} from "@medusajs/framework/http"
import { updateTranslationsWorkflow } from "@medusajs/medusa/core-flows"

export async function POST(
  req: MedusaRequest,
  res: MedusaResponse
) {
  const { result } = await updateTranslationsWorkflow(req.scope)
    .run({
      input: {
        selector: { reference_id: "prod_123", locale: "fr-FR" },
        update: { translations: { title: "Nouveau titre" } }
      }
    })

  res.send(result)
}
```

:::note

:::
