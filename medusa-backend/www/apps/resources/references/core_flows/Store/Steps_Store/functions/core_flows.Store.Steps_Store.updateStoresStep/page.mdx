# updateStoresStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateStoresStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates stores matching the specified filters.

## Example

```ts
const data = updateStoresStep({
  selector: {
    id: "store_123"
  },
  update: {
    name: "Acme"
  }
})
```
