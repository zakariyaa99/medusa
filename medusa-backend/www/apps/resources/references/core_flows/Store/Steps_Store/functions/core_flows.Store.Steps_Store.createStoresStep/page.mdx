# createStoresStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createStoresStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates one or more stores.

## Example

```ts
const data = createStoresStep([{
  name: "Acme"
}])
```
