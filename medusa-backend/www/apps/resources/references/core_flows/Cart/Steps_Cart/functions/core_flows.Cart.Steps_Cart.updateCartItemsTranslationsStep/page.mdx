# updateCartItemsTranslationsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateCartItemsTranslationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step re-translates all cart line items when the cart's locale changes.
It fetches items and their variants in batches to handle large carts gracefully.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { updateCartItemsTranslationsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = updateCartItemsTranslationsStep({
      "cart_id": "cart_123",
      "locale": "{value}"
    })
  }
)
```
