# getVariantsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `getVariantsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves variants matching the specified filters.

## Example

```ts
const data = getVariantsStep({
  filter: {
    id: "variant_123"
  }
})
```
