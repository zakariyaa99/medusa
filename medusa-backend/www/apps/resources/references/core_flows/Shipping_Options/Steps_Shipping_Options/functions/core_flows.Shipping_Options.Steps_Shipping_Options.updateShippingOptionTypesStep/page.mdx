# updateShippingOptionTypesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `updateShippingOptionTypesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step updates shipping option types matching the specified filters.

:::note

This is available starting from [Medusa v2.10.0](https://github.com/medusajs/medusa/releases/tag/v2.10.0).

:::

## Example

```ts
const shippingOptionTypes = updateShippingOptionTypesStep({
  selector: {
    id: "sotype_123"
  },
  update: {
    label: "Standard"
  }
})
```
