# createRemoteLinkStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createRemoteLinkStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates links between two records of linked data models.

Learn more in the [Link documentation.](https://docs.medusajs.com/learn/fundamentals/module-links/link#create-link).

## Example

```ts
createRemoteLinkStep([{
  [Modules.PRODUCT]: {
    product_id: "prod_123",
  },
  blog: {
    post_id: "post_123",
  },
}])
```
