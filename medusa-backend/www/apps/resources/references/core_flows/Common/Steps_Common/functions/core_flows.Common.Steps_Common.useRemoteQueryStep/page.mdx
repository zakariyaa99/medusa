# useRemoteQueryStep - Medusa Core Workflows Reference

This documentation provides a reference to the `useRemoteQueryStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step fetches data across modules using the remote query.

Learn more in the [Remote Query documentation](https://docs.medusajs.com/learn/fundamentals/module-links/query).

:::note

This step is deprecated. Use [useQueryGraphStep](../core_flows.Common.Steps_Common.useQueryGraphStep/page.mdx) instead.

:::

## Example

To retrieve a list of records of a data model:

```ts
const products = useRemoteQueryStep({
  entry_point: "product",
  fields: [
    "*",
    "variants.*"
  ]
})
```

To retrieve a single item instead of a an array:

```ts
const product = useRemoteQueryStep({
  entry_point: "product",
  fields: [
    "*",
    "variants.*"
  ],
  variables: {
    filters: {
      id: "123"
    }
  },
  list: false
})
```

To throw an error if a record isn't found matching the specified ID:

```ts
const product = useRemoteQueryStep({
  entry_point: "product",
  fields: [
    "*",
    "variants.*"
  ],
  variables: {
    filters: {
      id: "123"
    }
  },
  list: false,
  throw_if_key_not_found: true
})
```
