# cancelOrderFulfillmentStep - Medusa Core Workflows Reference

This documentation provides a reference to the `cancelOrderFulfillmentStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step cancels an order's fulfillment.

## Example

```ts
const data = cancelOrderFulfillmentStep({
  order_id: "order_123",
  items: [
    {
      id: "item_123",
      quantity: 1
    }
  ]
})
```
