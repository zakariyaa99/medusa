# exportOrdersWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `exportOrdersWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow exports orders matching the specified filters. It's used to
export orders to a CSV file.

:::note

This workflow doesn't return the exported orders. Instead, it sends a notification to the admin
users that they can download the exported orders.

:::

<SourceCodeLink link="https://github.com/medusajs/medusa/blob/a960fb75c9f662372fd5cd623f883f9d6d8ad473/packages/core/core-flows/src/order/workflows/export-orders.ts#L89" />

## Examples

To export all orders:

<CodeTabs group="workflow-exection">
    <CodeTab label="API Route" value="api-route">
    
```ts title="src/api/workflow/route.ts"
import type {
  MedusaRequest,
  MedusaResponse,
} from "@medusajs/framework/http"
import { exportOrdersWorkflow } from "@medusajs/medusa/core-flows"

export async function POST(
  req: MedusaRequest,
  res: MedusaResponse
) {
  const { result } = await exportOrdersWorkflow(req.scope)
    .run({
      input: {
        select: ["*"],
      }
    })

  res.send(result)
}
```

To export orders matching a criteria:

<CodeTabs group="workflow-exection">
    <CodeTab label="API Route" value="api-route">
    
```ts title="src/api/workflow/route.ts"
import type {
  MedusaRequest,
  MedusaResponse,
} from "@medusajs/framework/http"
import { exportOrdersWorkflow } from "@medusajs/medusa/core-flows"

export async function POST(
  req: MedusaRequest,
  res: MedusaResponse
) {
  const { result } = await exportOrdersWorkflow(req.scope)
    .run({
      input: {
        select: ["*"],
        filter: {
          created_at: {
            $gte: "2024-01-01",
            $lte: "2024-12-31"
          }
        }
      }
    })

  res.send(result)
}
```
