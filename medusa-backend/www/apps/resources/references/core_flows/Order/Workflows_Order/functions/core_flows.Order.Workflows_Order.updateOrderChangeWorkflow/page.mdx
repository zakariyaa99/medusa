# updateOrderChangeWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `updateOrderChangeWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow updates an order change.
If the `carry_over_promotions` flag is provided, it calls [onCarryPromotionsFlagSet](../core_flows.Order.Workflows_Order.onCarryPromotionsFlagSet/page.mdx)
to handle the promotion update logic. Otherwise, it updates the order change directly.

You can use this workflow within your customizations or your own custom workflows, allowing you to wrap custom logic around
updating an order change and conditionally handling promotion carry-over.

:::note

This is available starting from [Medusa v2.12.0](https://github.com/medusajs/medusa/releases/tag/v2.12.0).

:::
