# addImageToVariantsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `addImageToVariantsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step adds an image to one or more product variants.

:::note

This is available starting from [Medusa v2.11.2](https://github.com/medusajs/medusa/releases/tag/v2.11.2).

:::

## Example

```ts
const data = addImageToVariantsStep({
  image_id: "img_123",
  add: ["variant_123", "variant_456"]
})
```
