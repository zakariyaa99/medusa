# removeImageFromVariantsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `removeImageFromVariantsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step removes an image from one or more product variants.

:::note

This is available starting from [Medusa v2.11.2](https://github.com/medusajs/medusa/releases/tag/v2.11.2).

:::

## Example

```ts
const data = removeImageFromVariantsStep({
  image_id: "img_123",
  remove: ["variant_123", "variant_456"]
})
```
