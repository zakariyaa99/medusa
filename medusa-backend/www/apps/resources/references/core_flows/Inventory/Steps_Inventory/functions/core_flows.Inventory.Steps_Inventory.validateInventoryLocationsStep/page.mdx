# validateInventoryLocationsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateInventoryLocationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step ensures that the inventory levels exist for each
specified pair of inventory item and location. If not,
the step will throw an error.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { validateInventoryLocationsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = validateInventoryLocationsStep({
      "inventory_item_id": "{value}",
      "location_id": "{value}"
    })
  }
)
```
