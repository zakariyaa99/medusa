# createStockLocations - Medusa Core Workflows Reference

This documentation provides a reference to the `createStockLocations`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates one or more stock locations.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { createStockLocations } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = createStockLocations({
      "name": "Eli"
    })
  }
)
```
