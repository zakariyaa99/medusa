# listFulfillmentProviders - Fulfillment Module Reference

This documentation provides a reference to the `listFulfillmentProviders` method. This belongs to the Fulfillment Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method retrieves a paginated list of fulfillment providers based on optional filters and configuration.

## Example

To retrieve a list of fulfillment providers using their IDs:

```ts
const providers = await fulfillmentModuleService.listFulfillmentProviders({
  id: ["sepro_123", "sepro_321"],
})
```
