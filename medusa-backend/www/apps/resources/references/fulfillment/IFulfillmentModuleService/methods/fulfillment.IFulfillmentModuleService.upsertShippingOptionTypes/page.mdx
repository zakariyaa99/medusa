# upsertShippingOptionTypes - Fulfillment Module Reference

This documentation provides a reference to the `upsertShippingOptionTypes` method. This belongs to the Fulfillment Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## upsertShippingOptionTypes(data, sharedContext?): Promise&#60;[ShippingOptionTypeDTO](../../../interfaces/fulfillment.ShippingOptionTypeDTO/page.mdx)&#62;

This method updates or creates a shipping option type if it doesn't exist.

### Example

```ts
const shippingOptions =
  await fulfillmentModuleService.upsertShippingOptionTypes({
    id: "sotype_123",
    label: "Express",
  })
```
