# updateStores - Store Module Reference

This documentation provides a reference to the `updateStores` method. This belongs to the Store Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateStores(id, data, sharedContext?): Promise&#60;[StoreDTO](../../../interfaces/store.StoreDTO/page.mdx)&#62;

This method updates an existing store.

### Example

```ts
const store = await storeModuleService.updateStores("store_123", {
  name: "Acme",
})
```
