# addImageToVariant - Product Module Reference

This documentation provides a reference to the `addImageToVariant` method. This belongs to the Product Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method is used to associate images and variants.

## Example

```ts
await productModuleService.addImageToVariant([
  {
    image_id: "img_123",
    variant_id: "variant_321",
  },
])
```
