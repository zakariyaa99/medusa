# batchVariants

This method manages the variants of a product. It sends a request to the
[Manage Variants](https://docs.medusajs.com/api/admin#products\_postproductsidvariantsbatch)
API route.

## Example

```ts
sdk.admin.product.batchVariants("prod_123", {
  create: [
    {
      title: "Blue Shirt",
      options: {
        Color: "Blue"
      },
      prices: []
    }
  ],
  update: [
    {
      id: "variant_123",
      title: "Pants"
    }
  ],
  delete: ["variant_123"]
})
.then(({ created, updated, deleted }) => {
  console.log(created, updated, deleted)
})
```
