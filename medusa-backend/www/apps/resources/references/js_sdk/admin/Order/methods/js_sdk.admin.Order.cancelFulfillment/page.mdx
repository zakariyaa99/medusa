# cancelFulfillment

This method cancels an order's fulfillment. It sends a request to the
[Cancel Fulfillment](https://docs.medusajs.com/api/admin#orders\_postordersidfulfillmentsfulfillment\_idcancel)
API route.

## Example

```ts
sdk.admin.order.cancelFulfillment(
  "order_123",
  "ful_123",
  {
    no_notification: false
  }
)
.then(({ order }) => {
  console.log(order)
})
```
