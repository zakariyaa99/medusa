# listChanges

This method retrieves a list of changes made on an order, including returns, exchanges, etc...

This method sends a request to the [List Changes](https://docs.medusajs.com/api/admin#orders\_getordersidchanges)
API route.

## Example

```ts
sdk.admin.order.listChanges("order_123")
.then(({ order_changes }) => {
  console.log(order_changes)
})
```
