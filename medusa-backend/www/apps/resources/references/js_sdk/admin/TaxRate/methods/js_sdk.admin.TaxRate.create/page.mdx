# create

This method creates a tax rate. It sends a request to the
[Create Tax Rate](https://docs.medusajs.com/api/admin#tax-rates\_posttaxrates)
API route.

## Example

```ts
sdk.admin.taxRate.create({
  name: "VAT",
  tax_region_id: "txreg_123",
  code: "VAT",
  rate: 2, // 2%
})
.then(({ tax_rate }) => {
  console.log(tax_rate)
})
```
