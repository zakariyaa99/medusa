# retrieve

This method retrieves a locale by its code. It sends a request to the
[Get Locale](https://docs.medusajs.com/api/admin#locales\_getlocalescode) API route.

## Example

To retrieve a locale by its code:

```ts
sdk.admin.locale.retrieve("en-US")
.then(({ locale }) => {
  console.log(locale)
})
```

To specify the fields and relations to retrieve:

```ts
sdk.admin.locale.retrieve("en-US", {
  fields: "code,name"
})
.then(({ locale }) => {
  console.log(locale)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/store#select-fields-and-relations).
