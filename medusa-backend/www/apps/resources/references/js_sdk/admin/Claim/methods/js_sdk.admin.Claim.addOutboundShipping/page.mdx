# addOutboundShipping

This method adds outbound an outbound shipping method to a claim. 
The outbound shipping method will have a `SHIPPING_ADD` action.

This method sends a request to the 
[Add Outbound Shipping](https://docs.medusajs.com/api/admin#claims\_postclaimsidoutboundshippingmethod)
API route.

## Example

```ts
sdk.admin.claim.addOutboundShipping(
  "claim_123", 
  {
    shipping_option_id: "so_123",
    custom_amount: 10
  },
)
.then(({ claim }) => {
  console.log(claim)
})
```
