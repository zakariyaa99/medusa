# delete

This method deletes a tax region. It sends a request to the
[Delete Tax Region](https://docs.medusajs.com/api/admin#tax-regions\_deletetaxregionsid)
API route.

## Example

```ts
sdk.admin.taxRegion.delete("txreg_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
