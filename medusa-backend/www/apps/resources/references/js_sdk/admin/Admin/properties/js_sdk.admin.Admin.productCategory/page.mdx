[Create Category](https://docs.medusajs.com/api/admin#product-categories\_postproductcategories)
API route.

### Example

```ts
})
.then(({ product_category }) => {
  console.log(product_category)
})
```

## list

This method retrieves a paginated list of product categories. It sends a request to the 
[List Product Categories](https://docs.medusajs.com/api/admin#product-categories\_getproductcategories) API route.

### Example

To retrieve the list of product categories:

```ts
sdk.admin.productCategory.list()
.then(({ product_categories, count, limit, offset }) => {
  console.log(product_categories)
})
```

To configure the pagination, pass the `limit` and `offset` query parameters.

For example, to retrieve only 10 items and skip 10 items:

```ts
sdk.admin.productCategory.list({
  limit: 10,
  offset: 10
})
.then(({ product_categories, count, limit, offset }) => {
  console.log(product_categories)
})
```

Using the `fields` query parameter, you can specify the fields and relations to retrieve
in each product category:

```ts
sdk.admin.productCategory.list({
  fields: "id,*products"
})
.then(({ product_categories, count, limit, offset }) => {
  console.log(product_categories)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/store#select-fields-and-relations).
