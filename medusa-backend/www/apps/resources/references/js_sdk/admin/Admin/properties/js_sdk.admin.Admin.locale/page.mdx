# locale - JS SDK Admin Reference

This documentation provides a reference to the `sdk.admin.locale` set of methods used to send requests to Medusa's Admin API routes.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## list

This method retrieves a paginated list of locales. It sends a request to the
[List Locales](https://docs.medusajs.com/api/admin#locales\_getlocales)
API route.

### Example

To retrieve the list of locales:

```ts
sdk.admin.locale.list()
.then(({ locales, count, limit, offset }) => {
  console.log(locales)
})
```

To configure the pagination, pass the `limit` and `offset` query parameters.

For example, to retrieve only 10 items and skip 10 items:

```ts
sdk.admin.locale.list({
  limit: 10,
  offset: 10
})
.then(({ locales, count, limit, offset }) => {
  console.log(locales)
})
```

Using the `fields` query parameter, you can specify the fields and relations to retrieve
in each locale:

```ts
sdk.admin.locale.list({
  fields: "code,name"
})
.then(({ locales, count, limit, offset }) => {
  console.log(locales)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/store#select-fields-and-relations).

## retrieve

This method retrieves a locale by its code. It sends a request to the
[Get Locale](https://docs.medusajs.com/api/admin#locales\_getlocalescode) API route.

### Example

To retrieve a locale by its code:

```ts
sdk.admin.locale.retrieve("en-US")
.then(({ locale }) => {
  console.log(locale)
})
```

To specify the fields and relations to retrieve:

```ts
sdk.admin.locale.retrieve("en-US", {
  fields: "code,name"
})
.then(({ locale }) => {
  console.log(locale)
})
```

Learn more about the `fields` property in the [API reference](https://docs.medusajs.com/api/store#select-fields-and-relations).
