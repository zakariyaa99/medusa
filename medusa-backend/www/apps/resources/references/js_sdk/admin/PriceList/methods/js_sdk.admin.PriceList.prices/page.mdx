# prices

This method retrieves the prices of a price list. It sends a request to the
[Get Prices](https://docs.medusajs.com/v2/api/admin#price-lists\_getpricelistsidprices)
API route.

:::note

This is available starting from [Medusa v2.12.3](https://github.com/medusajs/medusa/releases/tag/v2.12.3).

:::

## Example

```ts
sdk.admin.priceList.prices("plist_123")
.then(({ prices }) => {
  console.log(prices)
})
```
