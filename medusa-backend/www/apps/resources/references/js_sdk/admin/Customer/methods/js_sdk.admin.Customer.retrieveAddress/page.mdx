# retrieveAddress

This method retrieves a customer address by its ID. It sends a request to the
[Get Customer Address](https://docs.medusajs.com/api/admin#customers\_getcustomersidaddressesaddressid)
API route.

## Example

```ts
sdk.admin.customer.retrieveAddress("cus_123", "cus_addr_123")
.then(({ customer }) => {
  console.log(customer)
})
```
