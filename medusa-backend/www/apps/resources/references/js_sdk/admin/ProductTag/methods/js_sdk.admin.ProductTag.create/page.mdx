# create

This method creates a product tag. It sends a request to the 
[Create Product Tag](https://docs.medusajs.com/api/admin#product-tags\_postproducttags)
API route.

## Example

```ts
sdk.admin.productTag.create({
  value: "shirt"
})
.then(({ product_tag }) => {
  console.log(product_tag)
})
```
