# create

This method creates a return reason. It sends a request to the
[Create Return Reason](https://docs.medusajs.com/api/admin#return-reasons\_postreturnreasons)
API route.

## Example

```ts
sdk.admin.returnReason.create({
  value: "refund",
  label: "Refund",
})
.then(({ return_reason }) => {
  console.log(return_reason)
})
```
