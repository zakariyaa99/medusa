# locale - JS SDK Store Reference

This documentation provides a reference to the `sdk.store.locale` set of methods used to send requests to Medusa's Store API routes.

## list

This method retrieves the list of supported locales in the store. It sends a request to the
[List Locales](https://docs.medusajs.com/api/store#locales\_getlocales) API route.

### Example

```ts
sdk.store.locale.list()
.then(({ locales }) => {
  console.log(locales)
})
```
